# Enemy Chase Game

## Description

A simple third-person Unity game where an enemy chases and attacks the player. The player can move using WASD and control the camera using the mouse.

## Features

* Player movement (WASD)
* Third-person camera with mouse rotation
* Enemy AI (chase + attack)
* Health system with health bar
* Game over system

## Controls

* W A S D → Move player
* Mouse → Rotate camera

## How to Run

1. Open the project in Unity
2. Open SampleScene
3. Click Play

## Tech Used

* Unity 6
* C#
